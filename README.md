# Customer Segmentation Analysis

## 📌 Project Overview

Customer Segmentation Analysis is a machine learning project that groups customers based on their **Annual Income** and **Spending Score** using the **K-Means Clustering** algorithm. The objective is to identify distinct customer segments that help businesses understand purchasing behavior and design targeted marketing strategies.

This project demonstrates the complete data analytics workflow, including data preprocessing, exploratory data analysis (EDA), clustering, visualization, and interpretation of business insights.

---

## 🎯 Objectives

* Analyze customer demographic and spending data.
* Clean and preprocess the dataset for analysis.
* Explore customer income and spending patterns.
* Apply the K-Means clustering algorithm to identify customer segments.
* Visualize customer clusters using scatter plots.
* Interpret each cluster and provide actionable business recommendations.

## 📂 Dataset

**Dataset Used:** customer_segmentation Dataset

**Features:**

* Customer ID
* Gender
* Age
* Annual_Income
* Spending_Score

**Source:** Kaggle – Mall Customer Segmentation Dataset

## 📊 Project Workflow

1. Import required Python libraries.
2. Load the customer dataset.
3. Perform data cleaning and preprocessing.
4. Conduct Exploratory Data Analysis (EDA).
5. Select relevant features for clustering.
6. Determine the optimal number of clusters using the Elbow Method.
7. Train the K-Means clustering model.
8. Visualize customer segments.
9. Interpret cluster characteristics and generate business insights.

## 💡 Business Insights

The clustering process identifies customer groups such as:

* 💎 High Income – High Spending (Premium Customers)
* 💰 High Income – Low Spending (Potential Customers)
* 🛍️ Low Income – High Spending (Value Seekers)
* 📉 Low Income – Low Spending (Low Engagement Customers)
* ⚖️ Average Income – Average Spending (Regular Customers)

These insights can help businesses improve customer targeting, personalized marketing, loyalty programs, and resource allocation.
